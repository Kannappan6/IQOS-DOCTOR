# IQOS Device Diagnostics Demo

A very simple static web app for demonstrating a support-style diagnostics flow for IQOS devices.

## Run locally

Open `index.html` in a browser.

Optional local server:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## What it includes

- Local IQOS-style device image
- Simulated device connection
- Separate charger and holder diagnostic rows
- PMI and IBM demo branding
- Animated diagnostics progress bar
- Battery, heater, contact, and firmware checks
- Recommended action summary
- Optional support checklist

This is a sample demo app only and is not an official diagnostic tool.
